FAQ & Support
Quick answers for common questions — searchable knowledge base for all IDS staff.
Common questions
Where do I save project files?
All project files must be saved to Autodesk Construction Cloud (ACC) — not the local server. Use the IDS-2401 project hub and the ISO 19650 folder structure. Never save WIP files directly to a Published folder.
How do I name a Revit model correctly?
Use the ISO 19650-2 format: IDS-2401_IDS_ZZ_ZZ_MO_A_0001.rvt. The type code for a model is always MO. Use the Upload Document button in the File Register to auto-generate the correct name from metadata fields.
What is the latest drawing template?
Download the IDS Title Block Template v3.2 from the Collaboration section. Do not use old templates — always check the hub for the current version before starting a new drawing set.
How do I issue a revised drawing?
Increment the revision (P01 → P02…) in filename and title block. Upload to ACC WIP zone. Change suitability to S1 for review, then S2 to issue. Use ACC transmittal to notify recipients. Previous revisions are retained automatically.
What is the difference between S1 and S2?
S1 (Coordination) is for sharing with other disciplines for clash checking — NOT for client issue. S2 (Information) is a formal client issue. Always confirm with the project lead before moving from S1 to S2.
Who do I contact for BIM support?
Contact the IDS BIM Coordinator (S. O'Brien) or post in the Collaboration section. For urgent ACC access issues, contact IT Support. For training queries, use the Training Portal.
