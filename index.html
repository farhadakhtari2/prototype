FAQ & Support
Quick answers for common questions.
Common questions
Where do I save project files?
All project files must be saved to Autodesk Construction Cloud (ACC) — not the local server.
How do I name a Revit model correctly?
Use the ISO 19650-2 format: IDS-2401_IDS_ZZ_ZZ_MO_A_0001.rvt. Type code for models is always MO.
How do I issue a revised drawing?
Increment revision (P01 → P02…) in filename and title block. Upload to ACC WIP zone. Move to S1 for review, S2 for client issue. Use ACC transmittal to notify.
What is the difference between S1 and S2?
S1 is for cross-discipline clash checking only — not for client issue. S2 is a formal client issue. Always confirm with the project lead before changing.
Who do I contact for BIM support?
Contact the BIM Coordinator (S. O'Brien) or post in the Collaboration section. For ACC access issues, contact IT Support.
